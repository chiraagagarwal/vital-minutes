# 🚨 Vital Minutes

AI-powered emergency response system that converts unstructured panic input into structured real-time alerts.

---

## 🔥 Problem

In emergencies, people cannot clearly describe situations. Delayed or unclear communication costs lives.

---

## 💡 Solution

Vital Minutes allows users to:

* Send SOS alerts instantly
* Automatically classify emergencies using AI + rule engine
* Share GPS location
* Broadcast alerts to responders in real-time

---

## ⚙️ Features

* 🧠 AI + Rule-based emergency classification
* 📡 Real-time dashboard (Firebase)
* 📍 GPS tracking with Google Maps link
* 📢 Broadcast system (mass alerts)
* 🚨 Priority-based alert handling

---

## 🛠 Tech Stack

* HTML, CSS, JavaScript
* Firebase (Firestore + Hosting)
* Gemini API (AI classification)
